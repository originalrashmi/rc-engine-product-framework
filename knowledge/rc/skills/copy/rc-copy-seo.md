## Core Principle

**SEO must serve the user first.** Never sacrifice copy clarity for keyword placement.

---

## Keyword Types

| Type | Definition | Copy Application |
|------|-----------|-----------------|
| **Primary** | Main search term the page targets | H1, meta title, first paragraph |
| **Secondary** | Close variations | H2s, body copy |
| **Semantic** | Conceptually related terms | Throughout body copy naturally |
| **Long-tail** | Specific multi-word queries | FAQ, feature detail sections |

---

## On-Page SEO Elements

### Meta Title
- **Format**: `{Primary Keyword + Benefit} - {Brand}`
- **Length**: ≤60 characters
- Primary keyword near the beginning
- No keyword stuffing

### Meta Description
- **Format**: `{Problem/Hook} + {Solution/Benefit} + {CTA}`
- **Length**: ≤155 characters
- Each page gets a unique meta description

### Heading Structure

```
H1: Primary keyword + page promise (ONE per page)
  H2: Section with secondary keyword
    H3: Subsection detail
  H2: Another section with semantic keyword
```

- ONE H1 per page
- Never skip levels (H1 → H3 without H2)
- Headings must read naturally

---

## Content Rules

- Include primary keyword within the first 100 words
- Use 3-5 keyword variations naturally throughout
- Short paragraphs (2-3 sentences), bullet points, subheadings every 300 words
- Link to related pages with descriptive anchor text (not "click here")

---

## SEO Checklist

- [ ] Primary keyword in H1 naturally
- [ ] Primary keyword in meta title (first half)
- [ ] Meta title ≤60 characters
- [ ] Meta description ≤155 characters with CTA
- [ ] 3-5 semantic keyword variations in body
- [ ] Image alt text is descriptive
- [ ] Copy reads naturally - no keyword stuffing
- [ ] Each page has unique meta title and description
