# Examples

## sample-research-output/

Example output from running the RC Engine Pre-RC research pipeline on its own
"Post-RC Security Gate" feature. This demonstrates what the 20-persona research
phase produces:

- `brief.md` -- the original product idea
- `classification.md` -- Cynefin complexity assessment
- `persona-selection.md` -- which research specialists were activated
- `stage-1-meta/` through `stage-6-validation/` -- per-stage research artifacts
- `prd-post-rc-security-gate.md` -- synthesized Product Requirements Document
- `tasks-post-rc-security-gate.md` -- generated task list
- `RESEARCH-INDEX.md` -- token usage breakdown by specialist

This is read-only reference material. To generate your own research output,
run RC Engine on your product idea.
